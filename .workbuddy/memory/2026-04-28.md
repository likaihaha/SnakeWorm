### 功能设计
- **静止时生命感波动**：虫虫没有移动时，身体从头到尾产生正弦波小幅度波动
- **波动方向**：垂直于虫虫当前朝向

- **波动速度**：每秒 4 弧度（约 0.64 Hz）
- **波长参数**：段索引间距 0.6 弧度，形成从头到尾传递的波浪效果

### 实现细节
```javascript
// 在 Worm 构造函数中初始化
this.idleWavePhase = Math.random() * Math.PI * 2;

// 在 update 中更新相位
this.idleWavePhase += dt * 4;

// 在 draw 中应用偏移
const waveAmplitude = this.isMoving ? 0.3 : 2.5;
const v = this.velocity.normalize();
const perpX = -v.y;
const perpY = v.x;
const wave = Math.sin(this.idleWavePhase - i * 0.6) * waveAmplitude;
const seg = new Vector(segRaw.x + perpX * wave, segRaw.y + perpY * wave);
```

### 覆盖场景
- waitingForPlayer 等待玩家阶段：出场动画完成后继续波动
- AI 蚯蚓静止时：自动产生波动
- 玩家停止移动时：波动明显
- 玩家移动时：波动几乎消失（0.3px）

### 修复：头部抖动问题（三次修复）

## Bug修复：TypeError Cannot read properties of undefined (reading 'push')

### 问题
- 控制台报错：`Uncaught TypeError: Cannot read properties of undefined (reading 'push')`
- 位置：`Worm.update` 第1940行
- 原因：`Worm.update` 中使用了 `this.foods.push(food)`，但 `foods` 数组属于 `Game` 类，不属于 `Worm` 类

### 解决方案
- 将 `this.foods.push(food)` 改为 `allFoods.push(food)`（使用传入的参数）
- 粒子特效改为暂存到 `worm.pendingParticles`，由 `Game.update` 统一收集到 `this.particles`

### 代码变更
```javascript
// 修改前（错误）
this.foods.push(food);
this.particles.push(new Particle(...));

// 修改后（正确）
allFoods.push(food);
if (!this.pendingParticles) this.pendingParticles = [];
this.pendingParticles.push(new Particle(...));

// Game.update中添加收集逻辑
if (worm.pendingParticles && worm.pendingParticles.length > 0) {
    for (const p of worm.pendingParticles) {
        this.particles.push(p);
    }
    worm.pendingParticles = [];
}
```
