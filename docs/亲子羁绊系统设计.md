# SnakeWorm 亲子羁绊系统设计

> 创建时间：2026-05-03
> 视角：任天堂式情感设计

---

## 一、核心结论

> 当前的亲子系统是**"有"**，但还不是**"爱"**。

任天堂做过最好的亲子/羁绊系统（皮克敏、耀西、宝可梦、塞尔达的马），核心从来不是"机制有多复杂"，而是——

> **"你愿意为它承担风险。"**

---

## 二、任天堂的亲子设计哲学

| 原则 | 任天堂案例 | SnakeWorm 现状 |
|------|-----------|---------------|
| **它会死，所以你在意** | 皮克敏会淹死、被吃掉、被踩扁 | 幼体被敌人咬死已有基础 ✅（v1.67已实现死亡动画） |
| **它有性格，所以不无聊** | 每只宝可梦性格不同，耀西有不同颜色 | 当前幼体没有个性 ❌ |
| **它会回报你，所以值得投入** | 皮克敏帮你搬东西、打怪、开路 | 当前幼体只会跟着吃 ❌ |
| **它在成长，所以有期待** | 耀西从蛋里孵化，宝可梦进化 | 幼体→成年已有，但太简单 ❌ |
| **失去它会痛，所以珍惜** | 皮克敏全灭的动画、塞尔达的马死了 | 幼体死亡已有动画（v1.67） ✅ |

**核心洞察**：任天堂从来不靠"数值奖励"让你在意一个角色，靠的是**情感设计**——动画、音效、行为反馈。

---

## 三、当前亲子系统的问题诊断

```
当前幼体的行为：跟随 → 吃宝珠 → 长大 → 成年 → 独立（AI虫）

问题：
┌─────────────────────────────────────┐
│  跟随 ← 没有选择权，机械跟随        │
│  吃宝珠 ← 和你抢食物，是竞争关系     │
│  长大 ← 数值变化，没有仪式感         │
│  成年 ← 突然变成AI，情感断裂         │
│  死亡 ← 已有动画，但母体反馈不足     │
└─────────────────────────────────────┘
```

**最大的问题**：幼体在"和你抢资源"，而不是"帮你"。这违反了任天堂的第一原则——**羁绊应该让双方都受益**。

---

## 四、五阶段情感设计

### 🥚 阶段一：诞生——让出生变成"事件"（✅ 已实现 v1.67）

> 你的尾巴断了，碎裂成一段段残片。但其中一段开始发光、膨胀……然后，一条小虫从里面钻出来。它第一次看到你，停顿了一秒，然后歪歪扭扭地游向你。

**已实现**：
- 光球收缩 → 裂开 → 小虫伸出头 → 左右看看 → 游向母体（孵化动画 5 阶段）
- 出生时播放独特的音效（`playBirthChime()`：C5→E5→G5 上行琶音）
- 小虫第一次游向母体时，身体微微发亮——认亲闪光
- 母体获得 3 秒无敌（出生奖励）

**任天堂参考**：皮克敏从地里拔出来时，会先愣一下，然后跟着你跑。那个"愣一下"的瞬间，就是感情的起点。

---

### 👶 阶段二：成长——让它有"性格"（建议新增）

**设计**：给幼体一个**随机性格标签**，影响它的行为方式。

| 性格 | 行为特征 | 视觉表现 |
|------|---------|---------|
| 🟢 勇敢 | 主动靠近敌人、吃远处宝珠 | 身体偏亮、移动快 |
| 🔵 温柔 | 躲在母体身后、不抢食物 | 身体偏柔、移动慢 |
| 🟡 好奇 | 到处乱跑、探索远处再回来 | 身体有小光点闪烁 |
| 🔴 淘气 | 抢你正在追的宝珠、偶尔撞你 | 身体有小火花 |

**关键设计**：性格不是纯随机——

> **它会模仿你。**
>
> 如果你经常冲锋杀敌，幼体倾向于"勇敢"。如果你经常绕路躲避，幼体倾向于"温柔"。它在学你。

这个设计来自《旺达与巨像》的马——你的马会记住你的骑乘习惯。任天堂深谙一个道理：**"它像你"比"它听你的"更能建立情感**。

---

### 🤝 阶段三：互动——让它"帮你"而不只是"跟着你"（建议新增）

**当前问题**：幼体是**负担**（消耗你的食物、增加你的风险），不是**伙伴**。

**任天堂原则**：皮克敏为什么可爱？因为它们帮你搬东西、打怪、开路。**你保护它们，它们回报你。**

#### 1. 侦察兵（好奇心性格的幼体）

> 幼体主动游出你视野范围去探索，如果发现宝珠或敌人，会发出**特殊的叫声+身体闪烁**来告诉你方向。

- 视觉：幼体身体闪 3 下，然后朝宝珠方向短暂游去再回来
- 音效：清脆的"嘀嘀嘀"提示音
- 实际价值：帮助玩家定位远处宝珠，尤其在大地图+关卡制下非常实用
- **情感价值**："它在帮我找吃的！"

#### 2. 护卫（勇敢性格的幼体）

> 当敌人靠近你时，勇敢的幼体会主动挡在你和敌人之间，**替你挡一次攻击**，然后受伤变灰变慢，需要你喂它宝珠恢复。

- 受伤表现：身体颜色变灰、移动速度减半、尾部颤抖动画
- 恢复方式：母体靠近它，自动"喂"它 1 节身体（类似当前幼体进食的逆向机制）
- **情感价值**：它为你挡了刀，你必须照顾它——这是双向的付出

#### 3. 合击（成年后的后代）

> 成年后的虫虫不再是"路人AI"，而是可以和你进行**协同攻击**。

- 当你瞄准敌人射击时，成年后代会在敌人侧翼同时发射一颗子弹
- 当你冲刺时，成年后代会跟上形成**夹击阵型**
- 触发条件：你需要先"叫"它——靠近它并短暂停留 0.5 秒，它理解为"跟我来"
- **情感价值**："我们是一个团队"

#### 4. 撒娇（✅ 已实现 v1.67）

> 当你长时间没理它（距离过远超过 10 秒），幼体会发出低沉的"呜呜"声，身体光芒变暗。当你靠近它时，它会围着你转 3 圈，身体重新亮起来。

**已实现**：
- `sulkTimer` / `isSulking` / `sulkGlow` 状态系统
- 远离 300px 持续 10 秒 → 撒娇（身体变暗 alpha 0.3）
- 靠近 80px → 庆祝（围着母体头部转 3 圈，身体闪烁明亮）
- `playSulkSound()` / `playCelebrateSound()` 音效

这个纯情感机制，零游戏性价值，但**100% 任天堂式设计**——塞尔达的马在你靠近时会嘶鸣、皮克敏在你走过时会欢呼。这些反应不影响游戏数值，但让你觉得**"它在乎我"**。

---

### 💀 阶段四：死亡——让失去有重量（✅ 已实现 v1.67）

**当前**：幼体被咬死 → 3 秒死亡动画 → 灰色尸体沉底 → 光点消散。

**已实现**（v1.67）：

1. **flashing（0~1.5s）**：身体段从尾到头依次闪烁变灰
2. **turning（1.5~2.5s）**：整体变灰，头部眼睛平滑转向母体方向
3. **sinking（2.5~4s）**：下沉 30px + 缩小到 40% + 渐闭眼横线
4. **gone（4s+）**：光点消散，留下灰色尸体

**母体反馈**：
- 幼体死亡瞬间，母体获得 0.5 秒的**屏幕震动**（`triggerScreenShake(4, 0.3)`）
- 灰色波纹扩散效果（`createRipple`）
- 音效：低沉心跳停止声（`playDeathHeartbeat()`：80Hz 低频长音 0.5 秒）

**任天堂参考**：皮克敏全灭时，所有皮克敏发出集体惨叫，Olimar 摘下帽子低头。那个画面让无数玩家在之后的每一局都小心翼翼。

---

### 🎵 阶段五：音乐与情感——隐藏的线

这是任天堂最强的秘密武器，也是 SnakeWorm **最容易实现但最容易忽略**的。

**亲子主题音乐**：

当前音乐系统 v2 已经很强大了。建议在以下场景触发**专属亲子旋律**：

| 场景 | 音乐变化 | 情感 |
|------|---------|------|
| 幼体出生 | 轻柔的上行琶音（✅ 已实现） | 希望、温暖 |
| 幼体靠近你 | 现有旋律加入一个高八度的陪衬声部 | 陪伴感 |
| 幼体帮你找到宝珠 | 短促的叮叮两声 | 奖励感 |
| 幼体受伤 | 旋律降调，节奏变慢（✅ 已实现） | 紧张、心疼 |
| 幼体死亡 | 所有音乐暂停 1 秒，然后一个低音长音（✅ 已实现） | 沉默的重量 |
| 后代成年独立 | 完整演奏一段上升旋律 | 成就感、骄傲 |

**关键**：音乐不需要很多音符。《塞尔达：旷野之息》的钢琴只有几个音符就创造了巨大的情感冲击。SnakeWorm 的水晶音色天然适合这种"少即是多"的设计。

## 五、与关卡系统的联动

亲子系统和 Z 字形关卡结合，会产生更强的化学反应：

### 1. 后代是"钥匙"

某些关卡的门需要**特定数量的后代一起推**才能打开：
- 你需要养 3 只幼体到成年，然后带着它们一起到门前
- 这给了养育后代一个**明确的玩法目标**，而不只是"养着玩"

### 2. 后代是"活地图"

在自由回访时，你可以让成年后代**留在某个关卡区域**充当"驻守者"：
- 它会持续探索该区域，标记出你没发现的隐藏内容
- 你回到该区域时，它会游过来"汇报"——身体闪烁指向隐藏区域的位置
- **这让后代变成了你在这个世界里的"代理人"**，离开你也能发挥作用

### 3. 后代数量 = 能力上限

| 后代数量 | 解锁能力 |
|---------|---------|
| 0 | 基础玩法 |
| 1 | 幼体帮你侦察宝珠 |
| 2 | 成年后代合击攻击 |
| 3 | 可以打开"家族门"（特殊关卡通道） |
| 5 | 解锁"家族领地"——你的后代永久驻守的区域，有特殊奖励 |

## 六、设计红线——不要做什么

| ❌ 不要 | 原因 |
|--------|------|
| 把后代做成"数值工具" | "养后代是为了+10%伤害"会把情感变成算计 |
| 让后代太多 | 1~3 只是最佳数量，超过 5 只你就分不清谁是谁了 |
| 做复杂的培养系统 | 任天堂不做"喂食→升级→天赋树"，那不是这个风格 |
| 让后代可以交易/丢弃 | 一旦可以"处理"掉后代，它就不再是"家人"了 |
| 惩罚玩家不养后代 | 后代应该是"锦上添花"而不是"不养就打不过" |

## 八、最终愿景

> 任天堂的游戏设计大师宫本茂说过："我设计游戏时，先想的不是'这个游戏有什么机制'，而是**'玩家玩完之后会跟朋友说什么'**。"

如果亲子系统做好了，玩家会跟朋友说：

> *"我的小虫虫帮我挡了一刀，然后死了。我当时真的愣了3秒。"*

**这句话，比任何数值奖励都值钱。**
